== Running the application

```
conda create -n neo_llm python=3.11
conda activate neo_llm
poetry install
```

Then run `streamlit run bot.py to start the app on link:http://localhost:8501/[http://localhost:8501/^].

[source,sh]
streamlit run bot.py
